# Simulateur de Gestion de Crise Cyber (AI-Powered)

Une application web interactive conçue pour moderniser et dynamiser les exercices de gestion de crise cyber (Tabletop Exercises). Fini les mains courantes statiques sur Excel : cet outil offre une expérience immersive, stressante et imprévisible grâce à l'Intelligence Artificielle Générative.

## 🚀 Pourquoi cet outil ?
Contrairement aux supports traditionnels figés, ce simulateur place les équipes (CODIR, DSI, RSSI, Com) dans un environnement vivant où le scénario évolue en temps réel, intégrant la pression temporelle et médiatique indispensable à un entraînement réaliste.

## ✨ Fonctionnalités Clés

*   **Scénarios Générés par IA** : Créez instantanément des situations de crise uniques (Ransomware, APT, DDoS...) via l'API Google Gemini. Choisissez votre difficulté : de *Facile* (incident standard) à *Cauchemar* (compromission totale).
*   **Simulateur de Réseaux Sociaux** : Un fil d'actualité type "Twitter" réagit en temps réel à vos décisions. L'IA génère des tweets de journalistes, d'hacktivistes ou de clients mécontents pour tester votre communication de crise.
*   **Cockpit de Pilotage Unifié** :
    *   **Main Courante** : Historisation automatique des événements.
    *   **Plan d'Action** : Suivi des tâches et assignation aux pilotes.
    *   **Annuaire & Fiches Réflexes** : Accès rapide aux ressources critiques.
*   **Conseiller Stratégique Virtuel** : L'IA analyse la situation en continu pour suggérer des axes stratégiques et rappeler les obligations légales.
*   **Post-Crise & REX** : Génération automatique de rapports d'analyse et tableaux de bord pour le Retour d'Expérience.

## 🛠 Stack Technique
*   **Frontend** : Vanilla JavaScript, HTML5.
*   **Design** : Tailwind CSS (Responsive).
*   **IA** : Google Gemini API (GenerateContent).
*   **Architecture** : Single Page Application (SPA), fonctionne sans backend (Serverless).

## ⚙️ Configuration
Pour activer la sauvegarde Cloud (Firebase), un fichier de configuration spécifique est requis.
